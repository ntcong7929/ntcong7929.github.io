Hướng dẫn cài đặt và sử dụng CTOOL XWORLD trên Google Cloud Shell

1. Giới thiệu

Chào mừng bạn đến với hướng dẫn chi tiết về cách thiết lập và chạy tool CTOOL XWORLD trên Google Cloud Shell. Đây là một nền tảng miễn phí, mạnh mẽ và lý tưởng để bạn chạy các tác vụ từ xa mà không cần cài đặt phức tạp. Hướng dẫn này phù hợp cho người dùng trên mọi thiết bị, bao gồm PC, Android và iOS.

2. Cài đặt các thư viện cần thiết

Trước tiên, chúng ta cần cài đặt các thư viện Python mà tool yêu cầu.

  1. Mở Google Cloud Shell trên trình duyệt của bạn bằng cách truy cập https://shell.cloud.google.com. Hoặc bạn có thể tải và mở từ ứng dụng Google Cloud trên điện thoại:
    • **Android:** Google Play Store
    • **iOS:** Apple App Store
  2. Sau khi Cloud Shell khởi động, bạn có thể cần nhấn vào biểu tượng **>_** để hiển thị giao diện dòng lệnh. Khi giao diện hiện ra, bạn sẽ thấy dấu nhắc lệnh.
  3. Tại đây, bạn nhập lệnh sau và nhấn **Enter** để cài đặt các thư viện: 
    pip install requests bs4 colorama psutil

    Lệnh này sẽ tải xuống và cài đặt tất cả các thư viện cần thiết: requests, bs4, colorama và psutil.

3. Tải tool CTOOL XWORLD

Bây giờ, hãy tải file tool về máy chủ Cloud Shell bằng lệnh curl. 

curl -o CTOOL.py https://raw.githubusercontent.com/C-Dev7929/Tools/refs/heads/main/main-xw.py

Lệnh này sẽ tải file từ GitHub và lưu nó dưới tên là CTOOL.py trong thư mục hiện tại của bạn.

4. Cách chạy tool

Sau khi đã tải tool về thành công, bạn chỉ cần chạy nó bằng lệnh python. 

python3 CTOOL.py

Nếu mọi thứ đều đúng, tool sẽ bắt đầu chạy và bạn có thể sử dụng.

5. Lỗi thường gặp và cách khắc phục

Lỗi 1: command not found: pip

Lỗi này xảy ra khi hệ thống chưa cài đặt Python package manager (pip). 

sudo apt-get update && sudo apt-get install python3-pip

Cách khắc phục: Chạy lệnh này để cài đặt pip.

Lỗi 2: No module named 'requests'

Đây là lỗi khi một trong các thư viện cần thiết chưa được cài đặt thành công. 

pip install requests bs4 colorama psutil

Cách khắc phục: Chạy lại lệnh cài đặt thư viện ở bước 2.

Lỗi 3: Permission denied (Lỗi từ chối quyền)

Lỗi này xảy ra khi bạn không có quyền thực thi file CTOOL.py. 

chmod +x CTOOL.py

Cách khắc phục: Cấp quyền thực thi cho file bằng lệnh này.
