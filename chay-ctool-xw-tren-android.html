Hướng dẫn cài đặt và sử dụng CTOOL XWORLD trên Android (Termux)

1. Giới thiệu

Chào mừng bạn đến với hướng dẫn chi tiết về cách thiết lập và chạy tool CTOOL XWORLD trên Android bằng ứng dụng Termux. Termux là một trình giả lập terminal mạnh mẽ cho phép bạn sử dụng các công cụ dòng lệnh Linux trên điện thoại của mình.

2. Yêu cầu và cài đặt Termux

Đầu tiên, bạn cần tải và cài đặt Termux từ **F-Droid**, vì phiên bản trên Google Play Store không được cập nhật thường xuyên. Tải Termux từ F-Droid tại đây

Sau khi cài đặt, bạn mở Termux và chờ quá trình khởi động hoàn tất.

3. Cài đặt các thư viện cần thiết

Trong Termux, bạn nhập các lệnh sau để cài đặt Python và các thư viện cần thiết.

  1. Cập nhật các gói hệ thống: 
    pkg update
  2. Cài đặt Python và `pip` (trình quản lý gói của Python): 
    pkg install python
  3. Cài đặt các thư viện Python cho tool: 
    pip install requests bs4 colorama psutil

    Lệnh này sẽ tải xuống và cài đặt tất cả các thư viện mà tool yêu cầu.

4. Tải tool CTOOL XWORLD

Để tải file tool, bạn cần cài đặt `curl` và sau đó sử dụng nó để tải file.

  1. Cài đặt curl: 
    pkg install curl
  2. Tải file tool: 
    curl -o CTOOL.py https://raw.githubusercontent.com/ntcong7929/ALL-TOOL/refs/heads/main/CTOOL.py

    Lệnh này sẽ tải file từ GitHub và lưu nó dưới tên là CTOOL.py.

5. Cách chạy tool

Sau khi tải file thành công, bạn chỉ cần chạy nó bằng lệnh sau: 

python CTOOL.py

Nếu mọi thứ đều đúng, tool sẽ bắt đầu chạy.

6. Lỗi thường gặp và cách khắc phục

Lỗi 1: command not found: curl

Lỗi này xảy ra khi gói `curl` chưa được cài đặt trong Termux. 

pkg install curl

Cách khắc phục: Chạy lệnh này để cài đặt `curl`.

Lỗi 2: No module named 'requests'

Đây là lỗi khi một trong các thư viện cần thiết chưa được cài đặt thành công. 

pip install requests bs4 colorama psutil

Cách khắc phục: Chạy lại lệnh cài đặt thư viện ở bước 3.
