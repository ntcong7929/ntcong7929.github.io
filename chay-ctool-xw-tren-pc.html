Hướng dẫn cài đặt và sử dụng CTOOL XWORLD trên PC

1. Giới thiệu

Chào mừng bạn đến với hướng dẫn chi tiết về cách thiết lập và chạy tool CTOOL XWORLD trên máy tính cá nhân sử dụng Python 3.12 và Command Prompt (CMD). Hướng dẫn này sẽ giúp bạn dễ dàng cài đặt và sử dụng tool trên máy tính của mình.

2. Yêu cầu và cài đặt Python

Để chạy tool, bạn cần cài đặt **Python 3.12** trên PC của mình. Nếu bạn chưa có, hãy tải về từ trang web chính thức: Tải Python 3.12 tại đây

Khi cài đặt, hãy đảm bảo bạn tích chọn vào ô **"Add python.exe to PATH"** để có thể chạy lệnh Python từ mọi vị trí trong CMD.

3. Cài đặt các thư viện cần thiết

Mở Command Prompt (CMD) trên máy tính của bạn và nhập lệnh sau. Bạn có thể tìm kiếm "cmd" trong menu Start. 

pip install requests bs4 colorama psutil cryptography rich

Lệnh này sẽ tải xuống và cài đặt tất cả các thư viện mà tool yêu cầu.

4. Tải tool CTOOL XWORLD

Để tải file tool, bạn có thể dùng lệnh `curl` trong CMD. Nếu máy tính của bạn không có `curl`, bạn có thể tải file bằng trình duyệt. 

curl -o CTOOL.py https://raw.githubusercontent.com/C-Dev7929/Tools/refs/heads/main/main-xw.py

Lệnh này sẽ tải file từ GitHub và lưu nó dưới tên là CTOOL.py.

5. Cách chạy tool

Sau khi tải file thành công, bạn di chuyển đến thư mục chứa file đó trong CMD, sau đó chạy lệnh sau: 

python CTOOL.py

Nếu mọi thứ đều đúng, tool sẽ bắt đầu chạy.

6. Lỗi thường gặp và cách khắc phục

Lỗi 1: 'pip' is not recognized...

Lỗi này xảy ra khi Python hoặc pip chưa được thêm vào biến môi trường hệ thống (PATH).

Cách khắc phục: Chạy lại trình cài đặt Python và chọn "Repair". Hoặc bạn có thể chạy lệnh với đường dẫn đầy đủ: 

py -m pip install requests bs4 colorama psutil

Lỗi 2: No module named 'requests'

Đây là lỗi khi một trong các thư viện cần thiết chưa được cài đặt thành công. 

pip install requests bs4 colorama psutil

Cách khắc phục: Chạy lại lệnh cài đặt thư viện ở bước 3.
